# Online Code Editor with User Registration

This project is built using:
- Python
- HTML
- CSS
- JavaScript
- Eel
- SQLite

## Features
- Online Python Code Editor
- User Registration (Name & Mobile)
- SQLite database storage


---

## Supported Code
- Print statements
- Loops
- Conditions
- Functions
- Lists, tuples, dictionaries (without input)

---

## Not Supported
- input() function
- Infinite loops
- GUI programs

---

## How to Run
pip install eel  
python main.py
## Platform
Use VS Code, PyCharm for better result
